═══════════════════════════════════════════════════════════════
        🚀 الآن: ارفع الصفحة على GitHub
═══════════════════════════════════════════════════════════════

1️⃣  أنشئ Repository على GitHub:
    
    🌐 اذهب إلى: https://github.com/new
    
    ⚙️ الإعدادات:
    • Repository name: quiztree-page
    • Public (اختر هذا الخيار)
    • ❌ لا تضيف README أو .gitignore أو LICENSE

2️⃣  اربط المجلد المحلي بـ GitHub:
    
    📝 افتح Command Prompt في المجلد الحالي ونفذ:
    
    git remote add origin https://github.com/YOUR_USERNAME/quiztree-page.git
    
    ⚠️ هام: استبدل YOUR_USERNAME باسم حسابك في GitHub

3️⃣  ارفع الملفات:
    
    git push -u origin main

4️⃣  فعّل GitHub Pages:
    
    🌐 اذهب إلى:
    https://github.com/YOUR_USERNAME/quiztree-page/settings/pages
    
    ⚙️ الإعدادات:
    • Source: Deploy from a branch
    • Branch: main
    • Folder: / (root)
    • اضغط Save

   → استخدم Personal Access Token بدل Password
   → أنشئه من: https://github.com/settings/tokens

   → نفذ: git remote remove origin
   → ثم أعد الخطوة 2

❌ خطأ "Repository not found":
   → تأكد أنك أنشأت Repository باسم quiztree-page
   → تأكد من اسم المستخدم الصحيح
